记Global Game Jam 2020

Posted on 2020-02-02 In 碎碎念 Views: Valine:

没想到第一次参加Game Jam就是如此特别。

由于湖北的疫情不容乐观，于是只能把自己关在家里，不抱有去现场参加的希望了，后来Jam开始的那天想了解一下今年的题目，才发现今年国内全部改成了线上进行。
喜出望外之余开始构思，今年的题目是Repair，于是第一反应竟然是我在故宫修文物，想了想可能美术资源不是很好做，于是开始发散思维。
然后很不巧地，工作群里接到了后天要线上开会的消息，我擦，后天不就是提交截止吗，我还得准备开会材料，这玩个毛？
冷静下来之后，掐指算了算时间，也许还能挤出24小时，但是再考虑到肯定会被老妈赶去睡觉不让通宵，于是大概也就16小时吧。反复告诉自己独立游戏重要的是idea不是内容丰富性。
到第二天总算有点想法了，大概类似于解谜向，开始准备素材，画sprite。
然后又很不巧的，之前加入的为对抗新冠病毒的志愿者组织nCoV-Support断断续续有点事情，于是等到准备完美术素材都已经是晚上了。开始实现逻辑的时候发现，有一个很重要的特性实现不出来，于是开始动摇，开始怀疑这个idea到底可不可行。终于又挣扎了一会之后，放弃了，留下半成品都不算的东西。
后面看了一些别人的作品，有的在美术上就做得很精美，完全不像是两天做出来的，有的作品完美契合主题，实现也很简单，感觉我后面还是思路有点跑偏了，还有很多是以这次的病毒事件为启发的，创意满满。

由于是线上进行，然后自己又是solo，所以感觉氛围不是很浓烈。但是总结一下，这次虽然没能成功完成，还是收获了很多东西的。首先还是没有充分的准备，加上各种事情突然挤在同一时间点，就有点自乱阵脚了，做的时候也记挂着其他事情，没有办法全神贯注火力全开。然后队友还是很重要的，自己在美术上花了太多时间了，而且如果有队友的话可以一起头脑风暴，讨论出更多结果，或者及时拉住跑偏的方向，一个人solo确实是有很多局限性。当然自己还是太菜了，没有办法实现脑子里的想法，也许我下一次应该转型策划？（笑）
