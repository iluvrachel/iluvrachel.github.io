从这里开始

发表于 2017-03-04

从上大学开始就一直心心念念要开个博客写文章,然后一直在忙各种项目和比赛耽搁了.
等到现在大三了难得有些课余时间,加上之前又因为做项目的缘故开始接触前端,就想着能不能靠着自己的知识撸一个博客出来,然而又不想花钱买服务器,毕竟只是做着玩玩,就选择了github pages + hexo这个方案,然后再注册个.tk免费域名,done!

不过说来简单,其实还是花了不少时间去配置各种东西,从空空如也到正式开始舒服地写作,总共涉及了下列工具:

  • git
  • github账户
  • 域名
  • node.js
  • hexo框架
  • Sublime text3及Markdown相关插件
  • 必要时可能还需要科学上网

常用操作命令
新建：hexo new “文章名字”
发布：hexo d -g

总之,这个博客将作为一个新的开始,记录一些有的没的,技术文章也好,吐槽也好,都是生活的痕迹.
