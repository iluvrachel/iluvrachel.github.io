MyEclipseй…ҚзҪ®Maven

Posted on 2017-05-31 Edited on 2018-02-12 In зј–зЁӢзӣёе…і Views: Valine:

д»ҠеӨ©иҜ•дәҶдёҖдёӢз”ЁMyEclipseжқҘй…ҚзҪ®mavenпјҢеҸҜд»ҘиҜҙеҮ з»ҸжҢ«жҠҳпјҢзҪ‘дёҠзҡ„ж•ҷзЁӢдә”иҠұе…«й—ЁпјҢз„¶иҖҢеҫҲеӨҡе№¶дёҚйҖӮз”ЁдәҺжҲ‘зҡ„з”өи„‘пјҢжҖ»д№ӢжҠҳи…ҫдәҶиӣ®д№…пјҢеңЁиҝҷйҮҢи®°еҪ•дёҖдёӢгҖӮ

дёӢиҪҪmaven

й…ҚзҪ®зҺҜеўғеҸҳйҮҸ

жү“ејҖзі»з»ҹзҺҜеўғеҸҳйҮҸж–°е»әдёҖдёӘM2_HOMEе°Ҷи§ЈеҺӢеҗҺзҡ„Mavenеӯҳж”ҫи·Ҝеҫ„ж”ҫеңЁжӯӨпјҢз„¶еҗҺеңЁpathи·Ҝеҫ„дёӢеҠ е…Ҙ%M2_HOME%\binи·Ҝеҫ„
жЈҖжөӢжҳҜеҗҰй…ҚзҪ®жҲҗеҠҹпјҢжү“ејҖе‘Ҫд»ӨиЎҢиҫ“е…Ҙecho %M2_HOME%жҹҘзңӢM2_HOMEжҢҮеҗ‘зҡ„Mavenеӯҳж”ҫе®үиЈ…зӣ®еҪ•жҳҜеҗҰжӯЈ
зЎ®пјҢиҫ“е…Ҙmvn -vжҹҘзңӢжҳҜеҗҰиғҪжүҫеҲ°жӯЈзЎ®зҡ„mvnжү§иЎҢи„ҡжң¬

е»әз«Ӣжң¬ең°д»“еә“

еңЁжң¬ең°ж–°е»әж–Үд»¶еӨ№MavenRepositoryпјҢеңЁMavenRepositoryдёӢе»әз«Ӣж–°ж–Үд»¶еӨ№repo
дҝ®ж”№mavenи§ЈеҺӢзӣ®еҪ•дёӢconfж–Үд»¶еӨ№йҮҢзҡ„setting.xml
е°ҶжіЁйҮҠзҡ„localRepositoryж Үзӯҫи§ЈйҷӨжіЁйҮҠе№¶е°Ҷең°еқҖдҝ®ж”№дёәеҲҡеҲҡж–°е»әзҡ„ең°еқҖеҰӮдёӢ

е№¶е°Ҷж”№еҠЁеҗҺзҡ„setting.xmlеӨҚеҲ¶еҲ°д»“еә“зӣ®еҪ•MavenRepositoryдёӢ
з„¶еҗҺеҶҚжү§иЎҢе‘Ҫд»ӨиЎҢmvn help:system
mavenе°ұдјҡејҖе§ӢдёӢиҪҪдёҖеӨ§е ҶдёңиҘҝпјҢиҝҮзЁӢеҸҜиғҪеҫҲж…ўпјҢд№ҹеҸҜиғҪжҢәеҝ«

MyEclipseй…ҚзҪ®maven

еңЁMyEclipseдёӯзҡ„PerferencesиҝӣиЎҢеҰӮдёӢй…ҚзҪ®пјҢж·»еҠ иҮӘе·ұзҡ„Maven
Maven4MyEclipseвҖ“>installationвҖ“>add
UserSettingдҝ®ж”№дёәд№ӢеүҚж”№еҘҪзҡ„setting.xml

еҹәжң¬зҺҜеўғе°ұжҗӯеҘҪдәҶпјҢеҸҜд»ҘејҖе§ӢжҗһйЎ№зӣ®

е»әз«ӢMavenйЎ№зӣ®

жү“ејҖMyEclipseвҖ“> newвҖ“> web projectвҖ“> еӢҫдёҠadd maven support вҖ“> java version ж”№дёә1.7 вҖ“> finish
е·ҘзЁӢе»әз«Ӣе®ҢжҜ•
еңЁpom.xmlеҸҜд»Ҙж·»еҠ йңҖиҰҒзҡ„еҢ…пјҢз„¶еҗҺmavenдјҡиҮӘеҠЁдёӢиҪҪпјҢиҝҷиҝҮзЁӢдјҡжҜ”иҫғеҚЎжҜ”иҫғж…ўпјҲиҫЈйёЎз”өи„‘пјҢеҘҪжғіиҰҒssdпјүпјҢMyEclipseеҸідёӢи§’дјҡжңүдёӘиҝӣеәҰжқЎжҳҫзӨәеҪ“еүҚжӯЈеңЁеҝҷд»Җд№ҲпјҢеҸҜд»ҘзӮ№ејҖзңӢ
еҰӮжһңдёҖзӣҙжІЎеҸҚеә”е°ұеҸій”®е·ҘзЁӢйҖүжӢ©mavenвҖ“>updateвҖ“>еӢҫдёҠforce update

еҰӮжһңиғҪеңЁжң¬ең°д»“еә“зңӢеҲ°дёӢеҘҪзҡ„jarеҢ…пјҢйӮЈеҹәжң¬жІЎй—®йўҳдәҶ
