MyEclipse配置Maven
Linzz
  2017-05-31 21:37:03   2024-10-04 19:27:50  
  • 编程相关
  • MyEclipse
  • maven
  

今天试了一下用MyEclipse来配置maven，可以说几经挫折，网上的教程五花八门，然而很多并不适用于我的电脑，总之折腾了蛮久，在这里记录一下。

下载maven

配置环境变量

打开系统环境变量新建一个M2_HOME将解压后的Maven存放路径放在此，然后在path路径下加入%M2_HOME%\bin路径
检测是否配置成功，打开命令行输入echo %M2_HOME%查看M2_HOME指向的Maven存放安装目录是否正
确，输入mvn -v查看是否能找到正确的mvn执行脚本

建立本地仓库

在本地新建文件夹MavenRepository，在MavenRepository下建立新文件夹repo
修改maven解压目录下conf文件夹里的setting.xml
将注释的localRepository标签解除注释并将地址修改为刚刚新建的地址如下

并将改动后的setting.xml复制到仓库目录MavenRepository下
然后再执行命令行mvn help:system
maven就会开始下载一大堆东西，过程可能很慢，也可能挺快

MyEclipse配置maven

在MyEclipse中的Perferences进行如下配置，添加自己的Maven
Maven4MyEclipse–>installation–>add
UserSetting修改为之前改好的setting.xml

基本环境就搭好了，可以开始搞项目

建立Maven项目

打开MyEclipse–> new–> web project–> 勾上add maven support –> java version 改为1.7 –> finish
工程建立完毕
在pom.xml可以添加需要的包，然后maven会自动下载，这过程会比较卡比较慢（辣鸡电脑，好想要ssd），MyEclipse右下角会有个进度条显示当前正在忙什么，可以点开看
如果一直没反应就右键工程选择maven–>update–>勾上force update

如果能在本地仓库看到下好的jar包，那基本没问题了
